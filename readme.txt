  实现的功能：
    1.爬取华工新闻网的信息，通过去除噪声等预处理操作，只采集网页的主题相关内容，先下载高质量和主题相关网页，保证高覆盖率；
    2.新闻关键信息采集：自动从网页中提取网页的时间、来源、核心图片等关键信息，其中时间包括网页更新时间以及新闻发布时间等
    3.新闻搜索平台：对采集的新闻以及关键信息建立索引，实现站内搜索，支持全文检索以及对关键信息的结构化检索；
    
  实现思路：
    1.新闻信息的爬取采用scrapy，scrapy相对于nutch来说，更灵活，更方便定制自己的爬虫。
    2.新闻的相关信息，保存在mysql数据库中。以后考虑移植到其他数据库上。
    3.考虑使用solr来对建立索引和站内搜索。
